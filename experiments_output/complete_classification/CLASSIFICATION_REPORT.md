# Complete Pattern Classification - All 114 Surahs
# تصنيف الأنماط الكامل - جميع السور الـ 114

## Classification System | نظام التصنيف

### Grade 0: No Heart (لا قلب)

**Criterion:** No closed upper curve, and no vertical symmetry

**المعيار:** لا منحنى علوي مغلق، ولا تناظر عمودي

### Grade 1: Primitive Heart (قلب بدائي)

**Criterion:** Closed upper curve, but open or asymmetrical bottom

**المعيار:** منحنى علوي مغلق، لكن قاع مفتوح أو غير متناظر

### Grade 2: Developed Heart (قلب متطور)

**Criterion:** Closed heart + simple internal symbol (e.g., dot, line)

**المعيار:** قلب مغلق + رمز داخلي بسيط (مثل: نقطة، خط)

### Grade 3: Integrated/Complete Heart (قلب متكامل)

**Criterion:** Symmetrical heart + clear internal symbol + strong thematic connection

**المعيار:** قلب متناظر + رمز داخلي واضح + ارتباط موضوعي قوي


## Distribution | التوزيع

- **Grade 1 - Primitive Heart**: 109 Surahs (95.6%)
- **Grade 2 - Developed Heart**: 5 Surahs (4.4%)
